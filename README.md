# h5games_web_mobileapp

Lightweight frontend project for aggregating games under the `mobileapp` directory. Built with `vite`.

Quick Start

```bash
cd h5games_web_mobileapp
npm install
npm run dev
```

Build & Preview

```bash
npm run build
npm run preview
```

Deployment

Deploy the build output (`dist/`) to Cloudflare Pages or any static host. After deployment ensure `games.json` is available at the site root or update `src/main.js` to point to the correct path.
